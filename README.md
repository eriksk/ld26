# Ludum Dare 26 Entry by Erik Skoglund (Skoggy)

## Installation

To run from source with bundler just clone the repo and run following commands

<pre>
  $ bundle install
</pre>

Tested with Ruby (MRI) 1.9.3

<pre>
  $ ./run
</pre>
